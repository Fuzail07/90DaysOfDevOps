Core directories : 

1. / (root) - every directories/ files comes under root.

2. /home - this is user's directory.

3. /root - this is a root user directory anything you create as a root user comes here.

4. /etc - every configuration file of linux resides here.

5. /var/log - logs of every application and system logs are generated here by default.

6. /bin - these contains binaries.

7. /usr/bin - binaries for user.

8. /opt - directory for third party appications like cloudera or any custom version.

9. /lib/ - it contains the libraries of every program . like kernel, grub,etc.
