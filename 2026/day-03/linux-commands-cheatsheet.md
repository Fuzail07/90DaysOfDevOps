Cheatsheet for Linux Commands:

Process management and file system management:-
ps - lists running process.
kill (PID)- kill process.
bg - lists background process. 
df - shows disk usage.
uname - shows system informantion.
scp - secured copy using ssh.
history - shows history of executed commands.
shutdown - use to power off system.
fsck - check for corrupted files.
find - use to find the files.
pwd - prints present directory.
lsof - lists open files.
vmstat - monitors memory performance. 
iostat - monitors input/output of the storage device.

Networking commands for troubleshooting: 
ping - it sends small packets to destination server to check connectivity.
dig - finds the dns,
netstat - check the network statistics and checks listening ports.
wget - use to download from the internet (but you have to install the package).
ifconfig - displays ip address (have to download nettools package).
